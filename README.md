## Run Locally

Clone the project

```bash
  git clone https://github.com/chadmuro/mainichi-habits.git
```

Go to the project directory

```bash
  cd mainichi-habits
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```

Open the app

```bash
  scan the QR code to run the app in Expo Go

  or

  press `i` to open iOS simulator
```
